# TechStore – Taller de Búsqueda Lineal

Este proyecto resuelve el **TALLER PRÁCTICO: BÚSQUEDA LINEAL** respetando todas las reglas y entregables.

## Estructura de archivos
- `funciones_busqueda.py`: Todas las funciones de búsqueda lineal (básicas, por diccionario y avanzadas).
- `datos_ejemplo.py`: Datos de ejemplo (productos y empleados) para pruebas.
- `sistema_tienda.py`: Menú interactivo de consola que integra todas las funciones.
- `informe_taller.pdf`: Informe con marco teórico, análisis de complejidad, respuestas a actividades y conclusiones.

## Requisitos
- Python 3.9+ (recomendado).
- No requiere librerías externas.

## Ejecución
```bash
python sistema_tienda.py
```

## Pruebas rápidas
En el menú principal:
1. Opción 1 prueba **búsqueda lineal simple** con contador.
2. Opción 2-4 cubren búsquedas por **productos**, **empleados** y **filtros**.
3. Opción 5 permite **búsqueda avanzada** (múltiples criterios, AND/OR y coincidencia aproximada).

## Complejidad
Todas las funciones basadas en búsqueda lineal son **O(n)** en tiempo y **O(1)** en espacio adicional. Ver detalles en el informe.

## Autoría académica
El código está ampliamente comentado para evidenciar la comprensión del algoritmo y facilitar su evaluación.
